# 乘积小于 K 的子数组

leetcode 713. (opens new window)

给你一个整数数组 nums 和一个整数 k ，请你返回子数组内所有元素的乘积严格小于 k 的连续子数组的数目。

示例1：

输入：nums = [10,5,2,6], k = 100
输出：8
解释：8 个乘积小于 100 的子数组分别为：[10]、[5]、[2],、[6]、[10,5]、[5,2]、[2,6]、[5,2,6]。
需要注意的是 [10,5,2] 并不是乘积小于 100 的子数组。

理解： num为nums第i位的值，包含num并且以num结尾的符合条件的数组数量为窗口右边界到i时的结果，那么遍历nums，使所有i位置的结果相加即为答案。

/**
 * @param Integer[] $nums
 * @param Integer $k
 * @return Integer
 */
function numSubarrayProductLessThanK($nums, $k) {
    $n = count($nums);
    
    $ride = 1;
    $ans = $left = $right = 0;
    
    while ($right < $n) {
        
        $ride *= $nums[$right];
        
        while ($ride >= $k && $left <= $right) {
            
            $ride /= $nums[$left];
            $left++;
            
        }
        
        // $nums[$right]
      	// $nums[$right] .. $nums[$right - 1]
      	// $nums[$right] .. $nums[$right - 1] .... $nums[$left]
        $ans += $right - $left + 1;
        $right++;
        
    }

    return $ans;
}
