# leetcode 04.06 (opens new window)

设计一个算法，找出二叉搜索树中指定节点的“下一个”节点（也即中序后继）。

如果指定节点没有对应的“下一个”节点，则返回null。

示例：

输入: root = [2,1,3], p = 1

  2
 / \
1   3

输出: 2

理解：

1、普通二叉树的下一个节点即中序遍历的下一个值；

2、题目给定二叉搜索树，根据其特点可知找到指定节点的最小的比它大的节点即可。

/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func inorderSuccessor(root *TreeNode, p *TreeNode) *TreeNode {
    cur := root
    var ans *TreeNode
    for cur != nil {
      if cur.Val > p.Val {
        ans = cur
        cur = cur.Left
      } else {
        cur = cur.Right
      }
    }
    return ans
}
